# Day 090 – Disappearing Text Writing App ✍️

## 📌 Overview
A writing application where text disappears if the user stops typing, encouraging continuous writing.

---

## 🎯 Features
- Timer-based text clearing
- Real-time typing tracking
- Focus enhancement

---

## 🧠 Concepts Used
- Tkinter
- Event handling
- Time-based logic

---

## ⚙️ Workflow
1. User starts typing
2. Timer resets on each keystroke
3. If idle → text clears

---

## 📍 Learning Outcome
- Event-driven programming
- Building behavioral applications
