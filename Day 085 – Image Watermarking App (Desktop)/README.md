# Day 085 – Image Watermarking App 🖼

## 📌 Overview
A GUI application that allows users to add watermarks to images.

---

## 🎯 Features
- Upload image
- Add custom watermark text
- Save modified image

---

## 🧠 Concepts Used
- Tkinter (GUI)
- PIL (Python Imaging Library)
- File handling

---

## ⚙️ Workflow
1. User uploads image
2. Adds watermark text
3. Program overlays text
4. Saves new image

---

## 📍 Learning Outcome
- Image processing basics
- GUI + file manipulation

---

⭐ HIGH — Real-world utility app
