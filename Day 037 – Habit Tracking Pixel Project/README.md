# Habit Tracker 📅

## 📌 Overview
Tracks daily habits using an external API service.

---

## 🎯 Features
- Log daily activities
- Visualize progress
- API-based storage

---

## 🧠 Concepts Used
- REST API (POST/GET)
- Data tracking

---

## ⚙️ Workflow
1. User logs habit
2. Data sent to API
3. Stored and visualized
