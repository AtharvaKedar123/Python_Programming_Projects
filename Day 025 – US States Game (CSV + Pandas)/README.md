# US States Game 🗺

## 📌 Overview
An interactive geography quiz where users guess US states displayed on a map.

---

## 🎯 Features
- Map-based visualization
- Real-time guessing
- Score tracking

---

## 🧠 Concepts Used
- Pandas (data handling)
- Turtle graphics

---

## ⚙️ How It Works
- User enters state name
- Correct guesses appear on map
- Game continues until all states guessed

---

## 📍 Learning Outcome
- Combining data + visuals
- Working with CSV datasets
