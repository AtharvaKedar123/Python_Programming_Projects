# Day 093 – Custom Web Scraper (Goodreads) 📖

## 📌 Overview
A web scraper that extracts book data from Goodreads.

---

## 🎯 Features
- Scrape book titles, ratings, authors
- Store extracted data
- Handle HTML parsing

---

## 🧠 Concepts Used
- Requests
- BeautifulSoup
- HTML parsing

---

## ⚙️ Workflow
1. Send request to webpage
2. Parse HTML structure
3. Extract required data
4. Store or display results

- Handling real-world web data
