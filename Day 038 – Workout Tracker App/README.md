# Workout Tracker 🏋️

## 📌 Overview
Tracks workouts using natural language input and logs them via API.

---

## 🎯 Features
- NLP input parsing
- Automatic calorie tracking
- Data logging

---

## 🧠 Concepts Used
- NLP APIs
- API integration

---

## ⚙️ Workflow
1. User inputs workout text
2. NLP API processes input
3. Data logged via another API
