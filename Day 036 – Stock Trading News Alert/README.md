# Stock News Alert 📈

## 📌 Overview
Monitors stock price changes and sends alerts when significant movements occur.

---

## 🎯 Features
- Stock price tracking
- News integration
- Alert system (SMS/Email)

---

## 🧱 Architecture
- Stock API → Price data
- News API → Related articles
- Notification system

---

## 🧠 Concepts Used
- API chaining
- Conditional alerts
- Data filtering

---

## ⚙️ Workflow
1. Fetch stock data
2. Check percentage change
3. Fetch related news
4. Send alert

---

## 📍 Learning Outcome
- Combining multiple APIs
- Event-based automation
