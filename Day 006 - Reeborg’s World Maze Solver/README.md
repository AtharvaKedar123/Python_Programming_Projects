# Maze Solver 🧭

## 📌 Overview
A program that solves a maze by navigating through paths using logical movement rules.

## 🎯 Features
- Automated navigation
- Pathfinding logic

## 🧠 Concepts Used
- Loops
- Conditions
- Algorithmic thinking

## ⚙️ How It Works
- Program follows rules to move through maze
- Avoids obstacles and finds exit

## 📍 Learning Outcome
- Problem-solving using algorithms
