# Secret Auction 🏆

## 📌 Overview
A bidding system that determines the highest bidder without revealing bids.

## 🎯 Features
- Hidden inputs
- Winner detection

## 🧠 Concepts Used
- Dictionaries
- Loops

## ⚙️ How It Works
- Users enter bids secretly
- Program stores bids and finds highest

## 📍 Learning Outcome
- Using dictionaries for data storage
