# Amazon Price Tracker 🛒

## 📌 Overview
Tracks product prices on Amazon and alerts when price drops.

---

## 🎯 Features
- Web scraping product price
- Email alerts

---

## 🧠 Concepts Used
- BeautifulSoup
- SMTP

---

## ⚙️ Workflow
1. Fetch product page
2. Extract price
3. Compare threshold
4. Send alert
