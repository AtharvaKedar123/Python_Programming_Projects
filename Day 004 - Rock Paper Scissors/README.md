# Rock Paper Scissors ✊✋✌️

## 📌 Overview
A Python implementation of the classic Rock-Paper-Scissors game against the computer.

## 🎯 Features
- Random computer choices
- Win/lose/draw logic
- Replay capability

## 🧠 Concepts Used
- Random module
- Lists
- Conditional logic

## ⚙️ How It Works
- User selects rock, paper, or scissors
- Computer randomly selects option
- Program compares and declares result

## ▶️ How to Run
python main.py

## 📍 Learning Outcome
- Implementing game logic
- Using randomness effectively
