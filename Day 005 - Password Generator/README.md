# Password Generator 🔐

## 📌 Overview
Generates strong and random passwords based on user preferences.

## 🎯 Features
- Customizable length
- Mix of letters, numbers, symbols
- Secure randomization

## 🧠 Concepts Used
- Random module
- Lists and strings
- Loops

## ⚙️ How It Works
- User specifies number of letters, numbers, symbols
- Program randomly generates password

## ▶️ How to Run
python main.py

## 📍 Learning Outcome
- Understanding randomness
- Creating secure logic systems
