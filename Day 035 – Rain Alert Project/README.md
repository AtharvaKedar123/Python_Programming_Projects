# Weather App 🌦

## 📌 Overview
Fetches real-time weather data from an API and displays key metrics.

---

## 🎯 Features
- Live weather updates
- Temperature, humidity, conditions
- Location-based data

---

## 🧱 Architecture
- API Layer → Weather API
- Data Layer → JSON parsing
- UI/Output → CLI or GUI

---

## 🧠 Concepts Used
- API calls
- JSON handling
- Data extraction

---

## ⚙️ Workflow
1. Send request to API
2. Parse response
3. Display formatted output

---

## 📍 Learning Outcome
- Real-world data integration
