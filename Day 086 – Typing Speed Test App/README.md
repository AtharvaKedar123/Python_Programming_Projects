# Day 086 – Typing Speed Test App ⌨️

## 📌 Overview
A GUI-based application that measures typing speed and accuracy.

---

## 🎯 Features
- Timer-based typing test
- Words per minute (WPM) calculation
- Accuracy tracking

---

## 🧠 Concepts Used
- Tkinter
- Time module
- String comparison

---

## ⚙️ Workflow
1. Display text to user
2. Start timer
3. Track typed input
4. Calculate WPM & accuracy

---

## 📍 Learning Outcome
- Building performance-based applications
