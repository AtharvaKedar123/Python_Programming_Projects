# Coffee & WiFi Finder ☕📶

## 📌 Overview
A web application that allows users to find and add cafes with WiFi availability.

---

## 🎯 Features
- Add new cafes
- View cafe listings
- Store data in database

---

## 🧱 Architecture
- Flask backend
- SQLite database
- HTML templates (Jinja)

---

## 🧠 Concepts Used
- Flask routing
- Database integration
- CRUD operations

---

## ⚙️ Workflow
1. User submits cafe details
2. Data stored in DB
3. Displayed dynamically

---

## 📍 Learning Outcome
- Full-stack CRUD app design
