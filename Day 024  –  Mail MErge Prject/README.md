# Mail Merge 📧

## 📌 Overview
A file automation project that generates personalized letters by replacing placeholders in templates.

---

## 🎯 Features
- Reads names from file
- Replaces placeholders dynamically
- Generates multiple output files

---

## 🧠 Concepts Used
- File handling (read/write)
- String replacement

---

## ⚙️ How It Works
1. Read list of names
2. Load template file
3. Replace placeholder with name
4. Save new file

---

## 📍 Learning Outcome
- Automating repetitive tasks
- Working with file systems
