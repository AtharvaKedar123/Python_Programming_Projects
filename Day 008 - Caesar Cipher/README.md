# Caesar Cipher 🔑

## 📌 Overview
Encrypts and decrypts text using shift-based cipher logic.

## 🎯 Features
- Encode and decode messages
- Custom shift values

## 🧠 Concepts Used
- String manipulation
- Loops

## ⚙️ How It Works
- Each letter is shifted by a fixed number
- Reverse process decrypts message

## 📍 Learning Outcome
- Understanding basic encryption techniques
