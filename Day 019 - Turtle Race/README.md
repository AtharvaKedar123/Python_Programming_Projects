# Turtle Race 🐢

## 📌 Overview
Simulates a race between multiple turtles using graphical animation.

## 🎯 Features
- Multiple participants
- Random movement
- Betting system

## 🧠 Concepts Used
- Turtle graphics
- Random module
- Loops

## ⚙️ How It Works
- Turtles move randomly
- First to finish wins

## 📍 Learning Outcome
- Event-based simulation
