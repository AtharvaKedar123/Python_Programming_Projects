# Day 092 – Image to Color Palette List App 🎨

## 📌 Overview
Extracts dominant colors from an image and generates a color palette.

---

## 🎯 Features
- Upload image
- Detect dominant colors
- Generate color palette list

---

## 🧠 Concepts Used
- Image processing
- Color extraction
- Data clustering (optional)

---

## ⚙️ Workflow
1. Load image
2. Analyze pixel data
3. Extract dominant colors
4. Display palette

---

## 📍 Learning Outcome
- Understanding image analysis
- Working with pixel data
