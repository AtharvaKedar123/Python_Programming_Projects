# Quiz Project 🧠

## 📌 Overview
A quiz application that asks multiple-choice questions and tracks score.

## 🎯 Features
- Question bank
- Score tracking
- Sequential questioning

## 🧠 Concepts Used
- Classes (Question, QuizBrain)
- Lists
- OOP structure

## ⚙️ How It Works
- Questions loaded into system
- User answers one by one
- Score calculated

## 📍 Learning Outcome
- Designing class-based applications
