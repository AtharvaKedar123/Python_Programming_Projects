# Pong Game 🏓

## 📌 Overview
A two-player Pong game built using Turtle graphics.

## 🎯 Features
- Paddle movement
- Ball physics
- Score tracking

## 🧠 Concepts Used
- Classes (Ball, Paddle, Scoreboard)
- Collision detection

## 📍 Learning Outcome
- Building interactive games
- Object-oriented design
