# Pong Game 🏓

## 📌 Overview
A classic Pong game recreated using Python Turtle graphics with two-player gameplay and physics simulation.

---

## 🎯 Features
- Two-player paddle controls
- Ball movement with bounce physics
- Score tracking
- Game restart logic

---

## 🧱 System Design
- `Paddle` → Player-controlled object
- `Ball` → Handles movement & collision
- `Scoreboard` → Displays scores

---

## 🧠 Concepts Used
- OOP design
- Collision detection
- Real-time animation loop

---

## ⚙️ How It Works
- Players control paddles via keyboard
- Ball bounces off paddles and walls
- Missing ball results in score for opponent

---

## 📍 Learning Outcome
- Physics simulation basics
- Event-driven programming
