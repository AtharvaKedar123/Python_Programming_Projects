# Day 075 - Google Trends Data Resampling 🔍

## 📌 Overview
Analyzes time-series data from Google Trends using resampling techniques.

---

## 🎯 Features
- Time-series analysis
- Data resampling (daily → monthly/yearly)
- Trend comparison

---

## 🧠 Concepts Used
- Pandas resampling
- Time-series indexing
- Rolling averages

---

## ⚙️ Workflow
1. Load time-series dataset
2. Convert to datetime index
3. Resample data
4. Analyze trends

- Understanding trend smoothing
