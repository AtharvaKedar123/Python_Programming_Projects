# Higher Lower Game 📊

## 📌 Overview
A game where players guess which entity has a higher value (e.g., followers).

## 🎯 Features
- Data comparison
- Score tracking
- Continuous gameplay

## 🧠 Concepts Used
- Dictionaries
- Functions
- Random selection

## ⚙️ How It Works
- Two options are presented
- Player guesses higher value
- Correct guess continues game

## 📍 Learning Outcome
- Working with structured data
- Building interactive logic
