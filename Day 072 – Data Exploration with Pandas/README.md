# Day 072 - Data Exploration with Pandas 📊

## 📌 Overview
Exploratory Data Analysis (EDA) using Pandas to understand structure, trends, and patterns in datasets.

---

## 🎯 Features
- Dataset loading and inspection
- Summary statistics
- Handling missing values
- Filtering and grouping data

---

## 🧠 Concepts Used
- Pandas DataFrames
- `.describe()`, `.info()`
- Filtering & grouping
- Data cleaning

---

## ⚙️ Workflow
1. Load dataset
2. Inspect structure
3. Clean missing/invalid data
4. Perform exploratory analysis

- Building data intuition
